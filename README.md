# Chronhr — Marketing Website

Public marketing site for [Chronhr](https://chronhr.com), an AML intelligence platform built for compliance analysts.

## Stack

- React + TypeScript (TSX)
- Vite
- Plain CSS with custom properties

## Getting Started

```bash
npm install
npm run dev
```

## Design

See [design.md](./design.md) for the full design system — fonts, colors, tokens, layout, hover interactions, and component notes.

## Pages

- `/` — Single-page landing
- `/about` — About
- `/team` — Meet the team
- `/pricing` — Pricing & ROI calculator
- `/get-started` — Contact form
